Guía de Uso del Panel de Monitoreo

Esta página describe cómo utilizar el tablero y qué significa cada dato mostrado.

Acceso al tablero

Ingresá a /dashboard/ para ver la lista de microservicios registrados. El botón Refrescar actualiza manualmente la información.

Tabla de servicios

  • Nombre: identificador legible del servicio.
  • Estado: valor reportado por Actuator (por ejemplo, UP o DOWN).
  • Versión: versión de la aplicación obtenida de los metadatos de registro.
  • CPU: barra de progreso con el porcentaje de uso del CPU del servidor donde se ejecuta el servicio.
  • Memoria: barra de progreso con el porcentaje de heap utilizado según las métricas de la JVM.
  • Disco: barra de progreso con el porcentaje de espacio utilizado en el disco.
  • Última Actualización: fecha y hora del último cambio de estado informado.

Gráfico de estados

Debajo de la tabla se muestra un gráfico de barras con la cantidad de servicios por estado. Sirve para obtener una vista rápida de cuántos servicios están activos o presentan problemas.

Si hacés clic en el nombre de un servicio accederás a una página con un resumen de la instancia y un enlace directo a su documentación OpenAPI.

Consejos adicionales

Al explorar cada instancia desde la consola principal de Spring Boot Admin podrás acceder a métricas detalladas, trazas y logs. Este tablero ofrece un resumen liviano para seguimiento diario.
